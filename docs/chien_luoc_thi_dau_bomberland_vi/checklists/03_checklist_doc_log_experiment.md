# Checklist ghi log experiment

## Thông tin cơ bản

- [ ] Tên experiment.
- [ ] Ngày chạy.
- [ ] Người chạy.
- [ ] Code version/commit.
- [ ] Agent version.
- [ ] Checkpoint path.

## Cấu hình train

- [ ] Thuật toán.
- [ ] Feature version.
- [ ] Reward version.
- [ ] Opponent pool.
- [ ] Seed.
- [ ] Số episode/match.
- [ ] Hyperparameter chính.

## Kết quả train

- [ ] Reward curve.
- [ ] Loss curve nếu có.
- [ ] Win rate train/eval.
- [ ] Average rank.
- [ ] Survival steps.
- [ ] Fallback rate nếu có.

## Kết quả test

- [ ] Test vs Random/Simple.
- [ ] Test vs BoxFarmer/Smarter.
- [ ] Test vs Tactical/Genius.
- [ ] Test vs bản cũ.
- [ ] Test mixed baseline.
- [ ] Benchmark time.

## Phân tích hành vi

- [ ] Agent chết chủ yếu vì gì.
- [ ] Agent có bỏ item không.
- [ ] Agent có đặt bom tự kẹt không.
- [ ] Agent có đứng yên vô lý không.
- [ ] Agent có attack hiệu quả không.
- [ ] Agent có bị reward hacking không.

## Quyết định

- [ ] Giữ checkpoint.
- [ ] Nộp checkpoint.
- [ ] Train tiếp từ checkpoint.
- [ ] Bỏ hướng này.
- [ ] Sửa reward/feature/rule.
